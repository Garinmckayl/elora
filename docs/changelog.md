# Changelog

All notable changes to Elora will be documented in this file.

## [0.9.0] - 2026-03-10

### Added
- Voice-first multimodal AI assistant with Gemini 2.0 Live API
- Real-time vision through device camera with proactive commentary
- 40+ integrated tools (email, calendar, web search, browser, code execution)
- Skill system: search, install, create, and publish reusable AI skills
- People memory with face recognition and relationship tracking
- Proactive engine with morning briefings and smart notifications
- Music generation with Google Lyria 2
- Image generation with Gemini Imagen
- E2B cloud sandbox for persistent code execution
- Restaurant search and reservation system
- SMS messaging integration
- Google Slides and Docs creation
- Multi-agent architecture with 5 specialized sub-agents

### Infrastructure
- Google Cloud Run deployment with auto-scaling
- Firestore for persistent memory and user data
- LiveKit integration for real-time audio streaming
- Terraform IaC for reproducible infrastructure
