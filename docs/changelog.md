# Changelog

All notable changes to Elora will be documented in this file.

## [0.9.1] - 2026-03-13

### Added
- push_to_github tool: commit and push code to GitHub directly by voice
- Ethiopian power outage work planner skill
- Elora-prefixed commit messages for AI-authored commits

- Music and image generation
- E2B cloud sandbox

- Google Cloud Run deployment
- Firestore for persistent memory
- LiveKit integration
- Terraform IaC
